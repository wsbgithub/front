石猴出世花果称王。 西游记第一回：石猴出世花果称王。 在浩瀚无垠的东胜神州有一座名山名曰花果山。此山挺拔险峻，四季如春，山间流水潺潺，鸟语花香，更有奇花异草，珍禽异兽，宛如人间仙境。 在花果山的最高处有一块古老的仙石，不知历经了多少岁月，吸天地之精华，纳日月之灵气，逐渐有了灵性 一日天朗气清，惠风和畅，仙石突然裂开，从中蹦跳出一个石猴来。这石猴一出世便眼运金光，射冲斗府，惊动了天上的玉皇大帝。玉帝忙命千里眼、顺风耳查看，得知是石猴出世，虽感奇异，却也未多加理会。 石猴生得毛脸雷公嘴，身材矮小却异常灵活。他睁开眼，只见四周花果飘香，心中大喜，便在这山中嬉戏起来。不久，他便发现了山中的猴群，便以自身的机智与勇敢赢得了猴群的信任与尊敬。 一日，猴群在水边嬉戏时，一只老猴提议：谁能跳过眼前的瀑布，探明瀑布后的秘密，便拜他为王。众猴面面相觑，无人敢应。这时石猴挺身而出，纵身一跃，跳入瀑布之中。瀑布之后竟别有洞天，原来是一个宽敞的石洞，洞中石桌、石椅一应俱全，宛如人间府邸。 石猴带着这一发现回到猴群，众猴欢呼雀跃，纷纷拜他为王，称他为"美猴王"。从此石猴便在花果山称王称霸，与众猴一起享受天伦之乐，日子过得逍遥自在。 然而好景不长，美猴王逐渐意识到生命的短暂与无常。他亲眼目睹了猴群中的老猴一个个老去死去，心中充满了对死亡的恐惧与无奈。为了寻求长生不老之术，他毅然决定离开花果山，踏上寻仙访道的征途。 美猴王穿越千山万水，历经重重困难，终于来到了西牛贺洲的灵台方寸山。在这里，他拜得了菩提祖师为师，开始了他的学艺生涯。祖师赐他姓名孙悟空，并传授他七十二般变化、筋斗云等神通广大的法术。孙悟空天资聪颖，勤奋好学，很快便掌握了这些法术，成为了一名法力高强的仙猴。 然而好景不长，孙悟空因卖弄神通，被菩提祖师逐出师门。虽然心中充满了不舍与遗憾，但他也明白自己必须离开这里，继续他的征途。
